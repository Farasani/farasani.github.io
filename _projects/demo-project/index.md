# Specs
- 168MHz STM32F405RGT6
- 5x UARTS serial ports
- I2C1
- 6-Axis ICM-42688-P
- MAX7456 OSD
- BMP280 Barometer
- 1GB BlackBox
- 6 PWM/DSHOT outputs
